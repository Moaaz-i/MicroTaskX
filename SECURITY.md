# Security Policy

## Supported Versions

We actively monitor and support the latest release of MicroTaskX. Please ensure you are running the most recent version before reporting a security issue.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of MicroTaskX seriously. If you discover a security vulnerability, please do not open a public GitHub issue. Instead, report it privately to help protect projects using this library.

To report a vulnerability:

1. Send an email to [EMAIL_ADDRESS].
2. Include a detailed description of the vulnerability, the environment it was found in, and steps or code to reproduce it.

Once received, we will:

- Acknowledge the receipt of your report within 48 hours.
- Work closely with you to understand and resolve the issue.
- Provide a fix in an upcoming release and give proper credit to the reporter.
